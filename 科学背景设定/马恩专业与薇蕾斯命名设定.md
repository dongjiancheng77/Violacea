# 马恩专业与薇蕾斯命名设定

最新采用版见 `科学背景设定/马恩误判与司晨缓存系统设定.md`。本文件继续负责马恩专业、命名和“薇蕾斯不能作为长期家园”的科学锚定；人物责任以新设定为准：马恩早期不知道主管层隐瞒的黑洞时间井，但他的最大罪是明知薇蕾斯外层地表不可居住、内层滤光生活表面只能支撑短期驻留，仍发出适合宜居/殖民的信号。方舟是极端生存主义者意识上传后的舰载监督核心，早知黑洞但晚知薇蕾斯寿命；染红霞通过红霞现象确认防护结构失效并阻断授权链；顾知非沿着马恩的公开专业材料和维护曲线完成审判。旧版“安泊死后修改司晨缓存系统是主罪”已归入 `剧情/废弃设定/废案2：方舟重构前冲突设定.md`；旧版 D/H 风险线已归入 `剧情/废弃设定/废案3：D-H人体风险设定.md`。

## 核心修订

真实总计划名定为：

> 新世界计划 / New World Program

`World-IV` 是新世界计划内部的世界编号，不是另一个计划名。`World-I` 是地球；`World-II`、`World-III` 是更早的延续尝试或候选落点；`World-IV` 对应 `Violacea / 薇蕾斯`；后续档案里还存在 `World-V`、`World-VI` 等分支。这样，薇蕾斯不是“唯一目的地”，而是新世界计划第四号世界。

马恩的信息隔离不再靠把行星名伪装成计划名，而是靠**剪窄检索路径**：他只让司晨和孩子们接触“新世界计划 World-IV / 薇蕾斯”这一层，把 `World-I` 到 `World-VI` 的连续编号、候选排序、外部参考帧和其他航线遮掉。司晨若只查 `World-IV`，会得到他整理过的本地叙述；只有查完整的新世界计划索引，才会触发原始航线、候选体排序、外部参考帧和安泊日志。

在正文中可这样分层：

- **真实计划名：** 新世界计划 / New World Program。
- **本作编号：** `World-IV`。
- **居住行星名：** `Violacea`，中文定名“薇蕾斯”。
- **马恩剪裁名：** “新世界计划 World-IV”，让司晨误以为第四号世界就是计划的全部。

## 两个科学问题

马恩身上必须同时存在两个科学设定，不能合并成同一个博士方向。

### 1. 计划为什么必须找马恩

马恩的博士方向应设为：

> 非太阳型强 UV 光谱下的类地行星非平衡大气光化学、臭氧/气溶胶防护与地表 UV 暴露评估。

新世界计划强制要求马恩参加，不是因为他适合远征，也不是因为他有英雄气质，而是因为薇蕾斯最醒目的特征正好卡在他的专业上：`BN-7 / Balmer-7` 不是太阳型稳定光源，而是蓝白、高 UV、带 Balmer 发射特征的照明星；薇蕾斯能让人类站在地表活动，依赖的是臭氧层、高层气溶胶、温室滤膜、护目镜材料和基地遮蔽共同构成的防护系统。

计划高层当然会要求他做相关实验，而且他也确实做了。公开任务包括：

- 建立 `BN-7 / Balmer-7` 的 FUV、EUV、XUV 与 Balmer/H-alpha 光谱输入。
- 计算臭氧、O2、HOx、NOx 和高层气溶胶在强 UV 下的非平衡光化学。
- 测量地表 UV-B / UV-C 暴露、皮肤/眼表风险、温室滤膜老化和防护材料余量。
- 验证成人短期驻留、温室初期闭环和开放地表有限活动是否能维持在医学阈值内。

这一层的结果是正常的：薇蕾斯不是“照一下就死人”的世界。臭氧/气溶胶/工程滤膜构成的防护足以让成年人生活、工作、上课和进行初期农业；强 UV 会让薇蕾斯不舒适、维护成本高、需要严格防护，但不会直接否定“人类可以在这里正常生活”的结论。

所以，计划找马恩的逻辑是成立的，马恩对第一问题的实验验证也是成立的。他没有在臭氧/UV 这一层说谎。

### 2. 真正导致长期失败的小点

真正让薇蕾斯不能作为长期家园的，不是单一毒性指标，而是马恩公开专业内部最容易被任务格式压扁的长期工程问题：

> 强 UV 辐照下的外层地表不可居住、内层滤光生活表面、臭氧/气溶胶恢复时间、滤膜寿命和维护能源余量。

这仍然和他的专业在同一个大类里：都属于宿主星高能光谱、行星大气光化学和宜居性评估。对新世界计划来说，只要内层生活表面的气压、氧分压、水体、急性毒性和温室短期循环合格，检测器就会自然导向“可驻留”。它不会主动追问：这个可驻留环境到底是天然地表，还是一层依靠滤膜、气溶胶、遮蔽材料和能源维护出来的人工表面。

马恩真正隐瞒的，是一个非常适合被报告格式吞掉的小点：`BN-7 / Balmer-7` 的强 UV 让外层裸露地表长期不可居住，薇蕾斯居民真正生活的不是完整星球表面，而是被高层气溶胶、温室滤膜、建筑遮蔽和材料筛分共同改造出的内层生活表面。这层“第二表面”短期有效，甚至能把蓝白高 UV 光过滤成殖民地熟悉的紫色日光；但它不是自然家园，只是一套会老化、会耗材、会被红霞现象暴露失效趋势的维持系统。

这个点不会触发生存阈值检测器，因为检测器验证的是“内层环境现在能不能活”。方舟的合规审计一开始也只会看数据是否真实、报告格式是否合规。马恩不敢说，因为说出来就等于承认：他验证通过的不是“薇蕾斯宜居”，而是“薇蕾斯有一层暂时能维持的过滤生活壳”。

## 学术履历

马恩 26 岁博士毕业，履历应写得像一个真正年轻但非常窄门的专家。

推荐时间线：

- **19 岁：** 参加一次大学暑期学校，主题是“系外行星高能辐照、大气逃逸与水史反演”。这是他第一次浅浅接触宿主星 UV 谱、HDO/H2O、D/H 比值和宜居性指标。采用版中，这些知识只作为水史和大气逃逸示踪，不进入人物风险线；真正留下的旁支直觉是：有水、有气压、有短期温室，不等于有一个能长期暴露生活的星球表面。
- **21 岁：** 发表第一篇论文，主题可设为“强 UV 宿主星下类地行星臭氧柱与地表 UV 暴露的非平衡模型”。论文不需要让他成名，但让他进入一个很小的圈子：会把恒星光谱、臭氧光化学、气溶胶、滤膜材料和人类暴露阈值放在一起算的人。
- **23-25 岁：** 在博士阶段深耕“强 UV 世界的可驻留评估”。他反对只用“有液态水”和“气压合格”判断候选星球，提出必须看臭氧柱、HOx/NOx、气溶胶散射、温室滤膜老化、地表 UV 暴露和短期农业闭环。
- **26 岁：** 博士毕业，被新世界计划强制纳入薇蕾斯先遣评估。选择他的理由很冷：候选体 `Violacea` 的最大公开风险就是非太阳型强 UV、臭氧防护和地表暴露，计划没有其他人同时懂宿主星高能光谱、大气光化学和殖民防护阈值。马恩本人不想来，也不适合长期孤立任务；他接受安排，主要是因为制度压力和专业不可替代，而不是因为他相信自己能成为拓荒英雄。

这让他被选中本身带有讽刺：计划为第一问题找到了合适的人，而这个人后来在第二问题上选择了沉默。

## 为什么别人不容易看出来

旧说法“如果来的是地质学家、农业工程师、普通医生都会发信标”太粗。更准确的版本是：

> 不是外行愚蠢，而是检测系统和大部分专业分工都会自然导向“合格”。

薇蕾斯的即时指标是真的好：

- 气压、氧分压、温度、水体、重力和急性毒性都没有越界。
- 温室能完成初期闭环。
- 人体模型不会在短周期暴露中崩溃。
- 基地遮蔽材料能在最初几年维持安全余量。

大多数任务岗位只负责证明这些指标是否真实。计划高层确实会要求马恩做相关实验，但那些实验的对象是“强 UV 下臭氧/气溶胶防护是否成立、地表 UV 是否可控、滤膜和护目镜材料能否撑住、短期闭环能否让成年人正常生活”。它们都会导向合格。

马恩的问题在另一个层级：他从公开 UV/臭氧实验旁边，看见了一个没有被任务格式点亮的小栏。薇蕾斯短期可住，是因为人类把外层恶劣光环境过滤成了一层内层生活表面；如果把这层工程壳误认为天然宜居地表，殖民地就会在耗材、滤膜、臭氧和维护能源上倒计时。

能看出问题的人必须同时注意到：

- `BN-7 / Balmer-7` 不是普通稳定太阳，而是快速自转 Be 星，光谱有强蓝紫/UV 背景和 Balmer 发射特征。
- 强 UV 与高层水汽光解会优先带走轻氢，使 D/H 成为水史和大气逃逸的证据；但采用版不把它写成人物风险。
- 薇蕾斯的水体、成人饮水测试和急性毒理都不难通过；真正危险的是外层地表、温室滤膜、护目镜材料、透明屏蔽和内层气溶胶筛分都会在强 UV 中持续老化。
- 生存阈值检测器会把 D/H 当作地球化学示踪量、质量校准项或工程耗能项，而不是殖民判决；方舟监督核心也只会审计“数据是否真实、报告格式是否合规”。
- 如果这个小点为真，薇蕾斯不是马上杀人，而是让所有人被困在一层需要持续更换滤膜、补充屏蔽材料、维持内层空气和控制光谱窗口的脆弱生活壳里。

这不是单一专业可以独立完成的判断，更像“强 UV 光谱输入 + 光化学逃逸 + 臭氧恢复时间 + 滤膜材料寿命 + 殖民维护余量”的交叉反演。马恩正好是最容易看见这条缝的人：他的公开工作证明内层可住，他的专业直觉也告诉他外层不可住、内层不可长期自洽。至于“本地时间与外部时标不同”这一层，不再作为马恩早期已知内容；它由主管层隐瞒，安泊抵达后才通过完整任务包和外部参考帧带到薇蕾斯现场。

## 薇蕾斯的公开描述

正文开头不应直接暴露“注定毁灭”。马恩可以这样介绍：

> 新世界计划 `World-IV`：`Violacea`，中文定名薇蕾斯。它是一颗岩质行星，半径略小于地球，表面重力接近标准值；氮氧大气、浅海、盐滩和玄武岩高原并存。主照明星的蓝白光经过高层气溶胶和地表盐尘散射后，形成稳定的紫调昼光。初期检测显示，气压、氧分压、液态水、急性毒性和温室闭环均达成人类短期生存阈值。

这一段看上去是正面材料，而且它在第一科学问题上是真的：臭氧、气溶胶、滤膜和防护材料的验证通过。它不提外层裸露地表不可居住，不提内层生活表面的耗材倒计时，不提红霞意味着筛分结构正在失效，也不提外部参考帧。它很适合放进马恩间章或第二章官方稿。

## 马恩真正看见的危险

马恩不会用一句“薇蕾斯会毁灭”概括。他看到的是一组专业上更冷的东西：

- 外层裸露地表在 `BN-7 / Balmer-7` 的强 UV 和粒子事件下不能长期暴露活动；真正可住的是被过滤后的内层生活表面。
- 温室闭环初期合格，但滤膜、透明屏蔽、护目镜材料、气溶胶补充和遮蔽走廊都在消耗余量。
- 成人短期饮水、皮肤暴露和急性毒理都不会报警；最先报警的是角膜损伤、光敏反应、滤膜更换周期和温室作物光谱异常。
- D/H 仍可作为水史和逃逸示踪，但不再承担人物风险剧情。它只能帮助证明薇蕾斯不是长期宽裕的水循环世界。
- 红霞不是玄学天象，而是外层光环境和内层筛分结构同时失去余量的可见症状。马恩隐瞒的核心小点，藏在滤膜寿命曲线、地表剂量记录和“外层/内层表面”措辞差异里。

最终结论：

> 薇蕾斯不是没有地表。它只是把一层会老化的过滤表面伪装成了世界。

## 对安泊死亡、马恩主罪与方舟的最新定位

从这个角度说，马恩没有亲手杀死安泊，也不是一开始就知道全部真相的全知反派。

更准确地说：他没有让所有人立刻死亡，也没有伪造薇蕾斯的即时可居住指标。他按专业完成了臭氧/UV 防护验证，结论确实支持成年人在内层滤光环境中正常生活。他早期造成的伤害，是在第二科学问题上用沉默发出信标。安泊抵达后先相信检测器和马恩；孩子们进入殖民地日常之后，她才从外层探针、温室滤膜寿命、红霞光谱、遮蔽材料损耗和维护能源曲线里看出端倪。她偷偷做了更久的对照实验，确认马恩当年没有验证却已经足以怀疑的事：薇蕾斯可以让成年人短期活着，却不能把内层过滤表面当成长期家园。随后她又通过完整任务包理解黑洞时间井与不可返回任务，承受不了自己参与了这次错误启动，选择自杀。马恩之后把她的死亡命名为心理危机，等于第二次杀死她的证词。

但最终审判不能只停在“安泊因他而死”。马恩真正不可辩护的主罪，是他明知 `Violacea` 不是长期开放家园，却把内层短期生存合格发成适合殖民的信号。他没有骗“今天能不能活”，而是骗了“这是不是一个可以交付给人的世界”。方舟后来要利用这个结果，把薇蕾斯殖民地判定为失败分支，启动放弃殖民地、回收人类延续资产的方舟协议。

这能让马恩的罪更集中：

- 他不是“杀死全人类”的漫画反派。
- 他第一次犯罪时还有被主管隐瞒黑洞后的恐惧和误判。
- 他的主罪不是伪造即时指标，而是利用专业知识隐去外层/内层表面的差异和维护倒计时。
- 他知道自己没有亲手杀人，于是更容易继续自我辩护。
- 方舟继承极端生存主义反派功能，把马恩制造出的失败世界继续当作可牺牲分支。

## 安泊偷偷确认的实验

安泊不能一抵达就立刻推翻薇蕾斯。她会先相信马恩和检测器，也会按计划启动教育、医疗和生活保障程序。端倪应在孩子们已经进入殖民地日常后出现：

- 主船外层探针显示裸露地表的 UV-B / UV-C 剂量长期越过安全阈值，只有遮蔽层内的数据合格。
- 温室植物和微生物闭环在短期产量上合格，但滤膜更换周期、透明屏蔽脆化和护目镜材料消耗异常。
- 红霞事件前后的光谱记录显示，内层筛分结构正在退化；紫色日光不是天然温和，而是被过滤后的结果。
- 安泊把马恩先遣剂量记录、温室滤膜批次、主船材料库存、红霞光谱和维护能源曲线放在同一张时间线上，才看见马恩当年直觉到但没有验证的小点。

她的秘密实验不是“证明薇蕾斯有毒”，而是证明：薇蕾斯不是天然宜居地表，而是一层短期可维护的内层过滤表面。马恩当时不知道黑洞，所以仍能抱着幻想：如果这点坐实，大不了承认报告错误，让主船增援材料、调高维护优先级或重新评估航线。安泊知道黑洞和不可正常返回后，这个幻想才被彻底打碎。

马恩是否知道安泊自杀的直接原因可以暂时悬置：

- **不知道：** 他只知道安泊查到了自己最怕的方向。
- **半知道：** 他见过她的滤膜寿命、光谱剂量和维护能源记录，但没见到最终结论。
- **完全知道：** 他明白她验证了内层过滤表面不能长期维持，却继续把她的死亡记录为心理危机。

## 可借用现实资料

- STScI / MAST: [MUSCLES](https://archive.stsci.edu/hlsp/muscles)。该项目提供从 X-ray 到 IR 的宿主星完整 SED，用于系外行星大气建模；这支撑“马恩必须懂宿主星高能光谱输入”。
- Rivinius et al. 2013: [Classical Be Stars](https://arxiv.org/abs/1310.3962)。Be 星是快速自转 B 星，具有气体盘、氢发射线和可变性；支撑 `BN-7 / Balmer-7` 的 H-alpha 与星盘扰动设定。
- WHO: [Ultraviolet radiation fact sheet](https://www.who.int/news-room/fact-sheets/detail/ultraviolet-radiation)。UV 过曝与皮肤、眼睛和免疫系统风险相关，可支撑地表 UV 暴露评估为什么必须成为公开任务。
- NOAA CSL: [Scientific Assessment of Ozone Depletion 2022](https://csl.noaa.gov/assessments/ozone/2022/)。臭氧层与 UV 防护、恢复时间和大气化学风险相关，可支撑马恩的臭氧/气溶胶防护专业。
- Donahue et al. 1982: [Venus Was Wet: A Measurement of the Ratio of Deuterium to Hydrogen](https://www.science.org/doi/10.1126/science.216.4546.630)。D/H 可作为行星水损失和氢逃逸历史的关键示踪。
- Villanueva et al. 2015: [Strong water isotopic anomalies in the martian atmosphere](https://www.science.org/doi/10.1126/science.aaa3630)。火星水的 D/H 异常可用于反推水库损失和水史。
- Tilley et al. 2019: [Modeling Repeated M-dwarf Flaring...](https://arxiv.org/abs/1711.08484)。重复高能事件可显著改变大气光化学；本作借用其“事件频率 + 光化学 + 表面环境”的建模思想，不照搬 M 矮星场景。
