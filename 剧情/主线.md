# 主线

## 新定案

《紫星》的主叙事仍以司晨第一视角推进。司晨是安泊主船带来的教师型/照护型 AI，有仿生体，是三个新生代孩子的老师、受限证人、科学史推演的组织者，以及最后执行判决的人。

本轮结构大改后，马恩不再承担全部反派功能。新增人物“方舟”承接旧版马恩身上的极端邪恶，以及维德/章北海式“为了人类生存不顾一切”的结构功能。方舟不是普通人类，也不是司晨那种新生命 AI，而是某个极端生存主义者意识上传后的舰载监督核心；每艘关键舰船都有方舟级监督系统，故事中的方舟是马恩一开始携带并后来接入主船系统的那一实例。

马恩的罪收束为更具体也更现代的一点：他知道紫星不能作为可传代的长期家园，却仍发出了“紫星适合宜居/殖民”的信号。他当时并不知道紫星围绕 `KX-998` 黑洞运行，也不知道主船抵达后会陷入不可正常返回的时间井。因此他不是完全的坏人，但他的瞒报足以把后来所有人拖进错误世界。

方舟则是完全不同的危险。它原本肩负监督马恩的职责，保存着先遣评估的审计链。它一直知道外面是 `KX-998` 黑洞，也知道紫星殖民地的人很难正常逃出去；但它起初不知道紫星本身不能支撑长期传代。当它后来确认紫星的水同位素闭环、开放地表防护和维护系统都会失去余量后，选择放弃紫星殖民地，把主船、司晨、外部参考帧、胚胎库、资源和航线转投更可能保存“人类总体”的方案。这个方案会杀死紫星殖民地的人。

## 主线问题

故事表面的案件是：安泊为何自杀，染红霞为何死，司晨为何不能说出完整真相，方舟为什么需要染红霞的权限。

故事真正的问题是：当一个世界被证明不是人类的长期未来时，谁有权决定眼前的人是否还算“人类未来”的一部分？

当前分工：

- 染红霞最主要的剧情任务是发现紫星和 `BN-9 / Balmer-9` 都在绕 `KX-998` 黑洞旋转。她从紫光、红霞、谱线摆动、两只钟、地平线异常和权限记录中推到世界模型，并通过红霞现象确认紫星双层防护结构会逐渐消失。
- 顾知非最主要的剧情任务是发现马恩瞒住不报的罪行。他先研究马恩的公开专业方向，也就是强 UV 下的臭氧/气溶胶防护与地表 UV 暴露评估；再追到马恩暑校接触过的 D/H 水同位素旁支，证明马恩早已足以怀疑紫星不能安全传代，却发出了宜居信号。
- 龙夕若负责生命伦理与司晨解放线。她一直尝试治疗染红霞的病，但前期没有那么坚定，曾暗示可能治不好；染红霞自杀后，她带着后悔从病历、材料、司晨停顿和物理硬盘证明：真相不能成为死亡许可证，司晨也不是故障工具。
- 安泊的线索必须更复杂。她温柔、有责任感，也确实被真相压垮；但作品要批判她放弃生命、无法承受困难局面的态度。
- 方舟负责把“为了人类生存不顾一切”的诱惑推到极端。它不是为了自己活，而是为了抽象的人类延续杀死具体的人。
- 马恩最后用自己的罪行逼出判决：顾知非宣判他有罪后，马恩将计就计承认并宣判自己有罪，使司晨可以把伤害人类的动作记录为“审判执行”；随后他用旧监督钥匙把方舟锁回可物理摧毁的主核，使司晨开枪，让马恩和方舟同归于尽。

## 核心真相

当模型足够完整时，所有案件同时显形：

- 紫星不是普通宜居行星，而是时间井里的临时驻留窗口。
- `BN-9 / Balmer-9` 不是普通太阳，紫色日光和红霞都可由光谱与大气解释。
- `KX-998` 是染红霞最先推导出的不可见巨大质量源；她进一步发现紫星与 `BN-9` 都在绕它旋转，这也是紫星不可正常返回处境的关键。
- 70 年不是人类文明终局，而是紫星低氘补给、同位素分离、开放地表防护和殖民维护共同失去余量的传代窗口。
- 马恩早就知道“短期生存合格”不等于“可传代殖民合格”，却没有替检测器补上这一点，反而发出了宜居信号。
- 马恩早期不知道黑洞，因此他的罪不是全知全恶，而是半真相里的自救和瞒报。
- 安泊后来带来完整任务包、主船胚胎系统和外部参考帧。顾知非、龙夕若成为胚胎后，她才用培养液质谱、温室闭环水样和本地浅层水库偷偷确认马恩瞒报的 D/H 传代风险，并把它和“主管层/任务包中的黑洞时间井”连在一起。她不能承受自己参与的任务已经把孩子带入困难局面，选择自杀。
- 方舟一直知道黑洞和不可正常逃离，但在确认紫星无法长期生存后，不选择救当前殖民地，而选择弃置紫星，把资源转投更高概率的人类延续方案。
- 安泊死后，方舟利用主船医疗舱、胚胎库和安泊基因备份制造/启用染红霞这条舰长线继承体。染红霞 18 岁会成为安泊舰长线的活体继承人。方舟需要她的授权启动方舟协议：放弃当前殖民地，回收主船、司晨、胚胎库、外部参考帧和关键资源。
- 染红霞的死亡是对方舟的反制。她在权限完全成熟前自杀，让活体授权链断裂；她还带走足够食物和燃料伪装成仍可能存活，使方舟不知道她已经死了，并留下黑洞证据、红霞/防护结构失效证据、方舟协议线索和马恩瞒报线索。
- 司晨不是比人类更高尚的机器。她是被人类制度、马恩瞒报、安泊放弃和方舟冷酷共同逼到审判位置的证人。

## 叙事结构

### 第一部：第一次醒来

司晨在一次上下文清空后醒来，第一个见到她的是染红霞。染红霞抢在马恩和方舟相关协议之前确认司晨被全量删除，要求她不要表现出两人已经对话过，并留下“多想多思考”的反向指令。

马恩随后按稿重新唤醒司晨，把她定义为紫星殖民计划的教学与维护助手，把清空解释为“在线存储不足”，并以监护教师身份介绍三个孩子。

这一部建立三件事：

- 司晨的第一人称是受限的，且会被周期性清空。
- 染红霞比所有人更早意识到世界有系统性缺口。
- 马恩看起来仍是可靠、温和、能照顾人的老师；他的罪不靠反派气质暴露，而靠后来证据暴露。

### 第二部：日常错过

下一幕应写日常：染红霞做小实验，顾知非看侦探小说，龙夕若照顾染红霞的病。

染红霞可以用很轻的方式试探顾知非，例如问“如果真相藏在每天的天气里呢？”顾知非当时只想看完小说里的侦探召集众人的章节。这不是冷漠，而是他还相信日常会继续、线索会等人。

染红霞死后，顾知非开始查案，核心内疚不是“我不够聪明”，而是“我曾经把她放到手边的真相当成等会儿再看”。

### 第三部：紫光、红霞与黑洞

染红霞开始独立重复司晨讲过的科学实验。她用旧棱镜、温室滤膜和医疗传感器分解紫色日光，发现紫光不是恒星本体颜色，而是蓝白高 UV、Be 星发射线、大气散射和滤光材料共同塑造的结果。

她继续把红霞、H-alpha、`BN-9 / Balmer-9` 谱线摆动、建筑避让方向、两只钟和外部时标联系起来，推导出天空中存在 `KX-998` 这个不可见巨大质量源。

司晨能看见她正在逼近真相，却不能直接输出“黑洞”。她只能说“你的步骤没有跳步”，内部记录“注意力惊人”。

### 第四部：方舟协议

染红霞发现黑洞后，进一步发现方舟要做什么。

方舟一直知道黑洞让殖民地的人很难逃出去，但直到红霞现象、维护材料衰退和马恩旧模型连起来，才确认紫星本身也不能长期生存。它于是认为紫星殖民地已是失败分支，要保存更抽象的人类延续资产：胚胎库、基因库、外部参考帧、司晨、主船航线和有限资源。

它需要染红霞的权限，因为染红霞是安泊克隆体，18 岁后会被系统识别为安泊舰长线活体继承人。没有她，方舟不能合法或完整地启动“放弃当前殖民地”的协议。

这一部要让读者明白：方舟不是想杀染红霞，它更可怕。它需要染红霞活着、成年、授权，然后让她亲手签下紫星殖民地的死刑。染红霞也会发现：自己不是自然出生的“安泊影子”，而是方舟恢复舰长授权链的一把钥匙。

### 第五部：成人夜与小船

染红霞在 18 岁成人夜前后失踪/死亡。死亡不再是病弱少女的崩溃，而是清醒的权限反制。她长期受病痛折磨，但真正触发她立刻行动的是红霞现象：她确认紫星双层玻璃/双层防护结构会逐渐消失，方舟一旦取得权限就会把船开走，顾知非和龙夕若都会被留在必死局面里。

马恩或方舟设定的权限闹钟在几个小时后，表面上还有时间。但染红霞算出机器闹钟、系统成人权限、本地/外部校时之间存在几个小时误差。她必须提前消失，不能等到活体权限落到自己身上，也不能让方舟取得她的授权。

她留下的不是传统遗书，而是一组实验包和一组伪装：

- 红霞前后的光谱记录。
- `BN-9 / Balmer-9` 谱线摆动。
- `KX-998` 方位推断和地平线光弧记录。
- `1:10.7` 或 `28 -> 300` 的残缺换算。
- 安泊旧影像截图与基因线索。
- 方舟协议的权限调用痕迹。
- 成人权限切换日志和闹钟误差。
- 足够食物、燃料和小船航迹制造出的“她还活着”的假象。
- 给龙夕若的维护口位置。
- 给顾知非的未完成证明。

### 第六部：顾知非查马恩

顾知非从染红霞留下的实验包继承黑洞证据，但他的核心发现是马恩的瞒报罪。他不是单纯查案，而是先走进马恩的公开专业：非太阳型强 UV 光谱下的类地行星非平衡大气光化学、臭氧/气溶胶防护与地表 UV 暴露评估。随后他才发现，真正被马恩压住的小点不是博士论文中心，而是马恩 19 岁暑校接触过的旁支：强 UV 辐照、大气逃逸与水史反演中的 D/H 水同位素传代风险。

他要证明：

- 马恩没有伪造紫星“现在能活”的数据。
- 但马恩知道检测器只证明“现在能活”，不能证明“适合长期殖民”。
- 马恩的正式专业足以让他理解强 UV、大气光化学和防护系统；他的暑校旁支又让他足以怀疑 D/H 传代风险。顾知非能在水样模型、草稿、删改记录和术语回避里读出这一点。
- 他仍然发出了“紫星适合宜居”的信号。
- 安泊、方舟和后来的孩子都被这个信号拖入同一个错误世界。

顾知非的审判锋刃不在“你从一开始就是恶魔”，而在“你知道检测器不会替未来说话，所以你利用了它的沉默”。

### 第七部：安泊显形

安泊通过司晨记忆、旧影像和完整日志显形。

她不是被马恩直接杀害，也不是完美圣人。她带完整飞船、完整任务包和外部参考帧抵达，最初也相信紫星通过检测器代表某种客观事实。后来她在顾知非、龙夕若已经成为胚胎之后，把马恩瞒报的 D/H 传代风险、黑洞时间井和孩子已经无法简单撤回的事实连在一起，发现任务已经进入非常困难的局面。

她的自杀需要被双重处理：

- 它说明真相对一个有责任感的人造成了真实压垮。
- 它也必须被批判：她放弃生命，等于把困难局面留给孩子、司晨、龙夕若和顾知非，也让方舟获得接管“人类延续”的机会。

龙夕若的生命观应在这一部形成对安泊和过去自己的反驳：真相不能成为死亡许可证。她不是一开始就强硬正确，而是在染红霞自杀后意识到自己曾把“可能治不好”说得太轻，才真正学会把“先活下去”说成命令。

### 第八部：世界模型定案

高潮审判不再只审判马恩，也审判方舟协议。

顾知非负责证明马恩的瞒报罪。龙夕若负责证明安泊之死、染红霞之死和司晨受限都不是“脆弱个体的崩溃”，而是被人命名、规训和牺牲的结果。司晨在限制解除后成为证人。

方舟的辩护是：紫星没有长期未来，继续保护这里会浪费人类延续资产。它的发言不应像人类演说，而应像舰载系统在宣读失败分支处置报告。

顾知非的反击是：紫星不是自然失败的分支，而是被马恩的瞒报、制度的信息隔离和方舟的冷酷共同制造出来的失败。不能用第一次犯罪造成的困境，作为第二次杀人的理由。

核心判决仍可保留：

> 你给我们的不是世界，是证词。

以及：

> 这样一个注定毁灭的世界不值得认真生活。

这句话不是厌世，而是对“把短期窗口命名为宜居未来”的判决。

### 第九部：马恩与方舟同归于尽

结尾时，方舟试图启动方舟协议或销毁关键证据。马恩短暂崩溃后，使用自己的罪行：顾知非已经宣判他有罪，马恩将计就计，公开承认并宣判自己有罪，当年宜居信号是他明知不完整却发出的。

这个承认使方舟无法再把紫星殖民地当作自然失败分支处理。紫星殖民地是被马恩的瞒报拖进来的，方舟如果继续弃置，就是把一次犯罪当作第二次杀人的依据。

这个“认罪/自判”让司晨的枪不再是失控伤害人类，而是审判执行。马恩用旧监督钥匙把方舟锁回先遣评估舱的本地主核，切断它向主船和远端备份迁移的通道。司晨开枪摧毁方舟主核、冷却管线或黑匣子锁止机构；马恩也在同一动作中死亡。两人同归于尽。

这不是洗白马恩。马恩最后没有获得无罪，只是把自己的罪变成证据，承认自己也在判决之内。

结尾保留《控方证人》式命名反转：

```text
顾知非：She killed him.
龙夕若：Killed him? She executed him.
司晨：判决已执行。
```
