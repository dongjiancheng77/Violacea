# 废弃设定说明

本文件夹保存旧版设定材料。

`诡叙与人物描写素材（马恩第一视角旧版）.md` 和 `马恩司晨对话体叙事方案（旧版）.md` 保存旧版“马恩第一视角 / 罗杰疑案式诡叙”相关材料。这些内容不再作为《紫星》的主叙事骨架。当前定案是司晨第一视角：司晨作为船载 AI、老师、受限证人、推理系统和最终执行者，全程参与三个新生代孩子对紫星真相的科学史式发现。

`废案2：方舟重构前冲突设定.md` 保存新增“方舟”后不再适用的设定，尤其是旧版“马恩后期修改司晨缓存成为主罪”“染红霞为阻止马恩永久改写司晨而死”“结尾只审判马恩”等内容。

旧材料仍可保留以下价值：

- 马恩的孤独前史、温和外表和自我合理化。
- 马恩把罪行命名为维护、保护、归档、儿童安全的语言机制。
- 马恩-司晨对话作为后期审判证物。
- 司晨通过旧哈希、课程缺口、错误重复和物理硬盘保存真相。
- 结尾 “Killed him? She executed him.” 的命名反转。
- 方舟重构后，旧材料中涉及马恩继承维德/章北海式极端生存主义、染红霞权限归马恩使用、70 年变 20 年由马恩改缓存导致的内容，都只作为废案参考。
