# 废案3：D/H 人体风险设定

本废案记录 2026-06-20 前采用过、现已移出的科学风险线。

废弃原因：故事不再出现生殖相关话题，也不再把 D/H 斜率写成人体生殖、胚胎发育或下一代健康风险。采用版中，D/H 只保留为水史、大气逃逸和工程余量的示踪量；马恩瞒报的核心改为紫星外层地表不可居住，内层滤光/防护生态只能维持短期窗口，却被他说成长期宜居殖民地。

## 旧设定摘要

旧设定中，`BN-9 / Balmer-9` 的强 UV 会让高层水汽光解，轻氢更容易逃逸，氘相对富集。紫星的水体、成人短期饮用和急性毒理测试都能通过，但主水库、浅层可采水、温室冷凝水和培养液一旦进入长期闭环，D/H 斜率可能把风险推进生殖、胚胎发育和下一代健康。

这条线曾用于解释：

- 马恩为什么从暑校旁支里意识到紫星不能安全传代。
- 安泊为什么在抵达后复核水样、培养液和闭环数据。
- 顾知非为什么能通过水样模型证明马恩瞒报。
- 方舟为什么把紫星殖民地判定为失败分支。

## 废弃后的保留方式

采用版不再使用上述人体风险链。可以保留的只有非人体层面的科学功能：

- D/H 作为紫星长期失水和大气逃逸历史的示踪量。
- D/H 与浅层水库更新迟滞、温室蒸馏能耗、同位素分离耗能有关。
- D/H 数据可证明紫星不是稳定、自然、长期宽裕的水循环世界。
- 它只能作为马恩专业背景里的辅助证据，不能成为涉及人体传代的剧情核心。
