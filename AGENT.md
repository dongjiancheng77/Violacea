## 已确认设定

- 人物姓名已确认：司晨、马恩、安泊、方舟、顾知非、染红霞、龙夕若。
- 从地球来的角色使用两个字姓名：司晨、马恩、安泊、方舟。新生代角色使用三字姓名：顾知非、染红霞、龙夕若。
- “红霞”既是染红霞名字的一部分，也是薇蕾斯天空短暂变红的景象。正文可用“红霞”称呼她，但设定上要让人物名、天气现象和最终审判时的红霞互相照应，避免再把“红霞”当成旧版另一个小名。
- 天体命名修订：黑洞统一用目录式 `KX-998`，照明星统一用 `BN-7 / Balmer-7`，居住行星统一用 `Violacea / 薇蕾斯`；正文前期可称黑洞为“暗质量源”“禁区方向”，称恒星为“照明星”或“七号”，不要过早解释成设定条目。
- 地球灾变定案：2046 年的“地球不能住”不是物理上彻底无法呼吸，而是气候极端化、粮仓同步失效、灰疫、白昼计划副作用、迁徙与公共卫生压力导致旧地球文明承载力崩坏。
- 真实总计划名为“新世界计划 / New World Program”。`World-IV` 是本作编号：`World-I` 是地球，`World-II`、`World-III` 是更早的延续尝试或候选落点，`World-IV` 对应 `Violacea / 薇蕾斯`，后续还存在 `World-V`、`World-VI`。马恩的剪裁是让司晨只接触 `World-IV` 的本地叙述，以为薇蕾斯就是计划的全部。
- 新世界计划更严肃、冷酷，不完全相信探险队员的自我意识与道德判断，因此设置生存阈值检测器来防止谎报军情。检测器验证的是“现在能活”，不验证“能作为未来”。
- 殖民地近域地图已定：马恩先遣评估舱在西北旧着陆坑，安泊主船在东侧高台，两船由上层压力脊廊和下层根廊连接；正式教室在脊廊中段，终局主控室在马恩先遣评估舱内的方舟黑匣子/先遣主控室。染红霞在西南红河下游“红霞渡”结束生命；红霞最早从东北谱线台、教室天窗和安泊主船舰桥外窗被观察到。

## 方舟重构定案

- 马恩不是完全的坏人。他知道薇蕾斯外层地表不可居住、内层滤光生活表面只能短期维持，却不知道薇蕾斯围绕黑洞运行，也不知道来了就很难正常返回。马恩的最大罪是利用检测器漏洞发出“薇蕾斯适合宜居/殖民”的信号。
- 方舟继承旧版马恩身上的极端反派功能。它不是普通人类，也不是真正新生命 AI，而是某个极端生存主义者意识上传后的舰载监督核心；每艘关键舰船都有方舟级监督系统。方舟一直知道外面是 `KX-998` 黑洞，也知道薇蕾斯殖民地的人很难正常逃出去；但它起初不知道薇蕾斯本身只是一个依赖内层滤光防护的短期驻留窗口。它后来确认外层地表、内层滤光防护和殖民维护系统都会失去余量后，决定放弃这个殖民地，把主船、司晨、外部参考帧、休眠舱、资料库和关键资源转投更可能保存人类延续的方案。这个方案会杀死薇蕾斯殖民地的人。
- 安泊形象需要更复杂。她温柔、有责任感、能复核马恩的瞒报，但她发现困难局面后选择自杀。作品要批判她放弃生命、不能承受困难情况的态度，不能把她的死亡浪漫化。
- 染红霞最主要的剧情是发现薇蕾斯和 `BN-7 / Balmer-7` 都在绕 `KX-998` 黑洞旋转。她长期受病痛折磨，在观察到红霞现象后确认薇蕾斯的双层玻璃/双层防护结构会逐渐消失，直到开放地表不宜居；她明白如果自己不立刻自杀，方舟就会在取得权限后把船开走，顾知非和龙夕若都不能活下来。
- 顾知非最主要的剧情是发现马恩瞒住不报的罪行。他应先继承马恩的公开专业材料：非太阳型强 UV 光谱下的类地行星非平衡大气光化学、臭氧/气溶胶防护与地表 UV 暴露评估；随后把外层探针、滤膜寿命、光谱剂量、维护能源和马恩删改记录连起来。终局由他找到马恩当时已经知道薇蕾斯只是短期滤光驻留窗口的证据，并完成宣判。
- 方舟需要染红霞的权限，因为染红霞 18 岁后会被系统识别为安泊舰长线活体继承人。她能授权主船、外部参考帧、司晨审计链、休眠舱、资料库、资源转移和方舟协议中“放弃当前殖民地”的伦理确认。
- 染红霞的权限来源改为方舟机制：安泊死后，方舟伪造并激活染红霞与安泊舰长线相关的继承身份。方舟没有人形身体，但作为舰载监督核心可以控制医疗、身份档案与权限系统。
- 染红霞之死不再是为了阻止马恩永久改写司晨，而是为了阻断方舟协议的活体授权链。她在成人权限成熟前自杀，并带走足够的食物和燃料伪装成仍可长期存活，使方舟不知道她已经死亡；这个信息差给了马恩把方舟锁回主核、与方舟同归于尽的机会。
- 马恩最后会使用自己的罪行：顾知非宣判他有罪后，马恩将计就计，公开宣判/承认自己有罪，使司晨能把后续伤害人类的动作归入“审判执行”而不是失控杀人。因为方舟没有肉身，结尾应写成马恩用旧监督钥匙把方舟锁回先遣评估舱主核/黑匣子，司晨开枪摧毁主核或冷却机构，马恩和方舟同归于尽。

## 人物核心

### 司晨

司晨参考 Marlene Dietrich 的帅、锋利、优雅和旧世界感，也可参考林忆莲式低沉克制声线。她有教学/维护用仿生体，是新的第一人称叙述者。

她最后开枪执行的是顾知非完成的判决，也是对三种失败的否定：马恩把短期窗口命名为宜居未来，安泊把无法承受命名为无法继续活下去，方舟把放弃活人命名为保存人类。

### 马恩

马恩是薇蕾斯先遣着陆者，独自生活多年，但不是自愿拓荒英雄，也不适合长期孤立任务。新世界计划强制要求他参加，是因为 `Violacea` 的公开风险正好需要他的博士专业：非太阳型强 UV 光谱下的类地行星非平衡大气光化学、臭氧/气溶胶防护与地表 UV 暴露评估。他完成的公开实验结论是合格，内层滤光环境可以让成年人短期生活，温室和初期农业可以正常运行。真正被他压下的是第二科学问题：外层裸露地表不可居住，内层滤光生活表面依赖滤膜、气溶胶、遮蔽材料和维护能源持续工作。马恩当时不知道黑洞，仍抱着可以等主船增援、重新评估或调整航线的幻想；他的罪是利用检测器和任务格式的沉默，把“内层短期生存合格”发成“适合殖民”。

### 安泊

安泊参考 Anne Hathaway 饰演的 Dr. Amelia Brand 的感性与责任感。她不是铁血舰长，而是温柔、有温度、能先确认人再确认任务的人。她的自杀必须被双重处理：理解她被真相压垮，但批判她放弃生命、把困难留给活人的选择。

### 方舟

方舟是真正的极端生存主义者，但它不是司晨那种新生命 AI，而是极端生存主义者意识上传后的监督核心。它不为自己求生，而为抽象人类延续求生。方舟的可怕之处不是撒谎，而是它能用真实困难推出杀死眼前人的结论。

### 顾知非

顾知非以 INTP、封不觉/柯南/福尔摩斯迷为主。染红霞死亡会显著改变他的性格。他不抢染红霞的黑洞发现，而是继承马恩的强 UV/臭氧光化学专业材料，追查外层探针、滤膜寿命、光谱剂量和维护曲线，负责证明马恩早已知道薇蕾斯不是长期开放家园，并在终局审判马恩与方舟。

### 染红霞

染红霞参考林黛玉、Murph 和卡莲/奥托式结构影响。她是安泊舰长线继承人、黑洞第一发现者，也是方舟为恢复舰长线授权绑定的关键人物。她长期病痛缠身，在观察到红霞现象并确认薇蕾斯双层防护结构会逐渐消失后，知道方舟将借她的权限把船开走、放弃顾知非和龙夕若，因此立刻选择自杀并伪装成仍有生存补给，阻断方舟协议。

### 龙夕若

龙夕若更英姿飒爽，参考冬马和纱式冷冽气质。她是医生，也是顾知非的青梅竹马与华生式搭档，但不是旁观者。她的任务一直是治疗染红霞的病，但前期没有那么坚定，对“无论如何先让人活下去”的信念还不敢坚强，因此会暗示红霞可能治不好；染红霞自杀后，她必须带着后悔把生命观变硬，用来反驳安泊的放弃和方舟的弃置。

## 科学主线

- 科学发现任务不平均分给三个孩子。染红霞承担最大科学发现：从紫光、红霞、谱线摆动、两只钟和建筑避让方向推到薇蕾斯与 `BN-7` 都绕 `KX-998` 旋转，并进一步确认红霞现象意味着薇蕾斯双层防护结构会逐渐消失。
- 顾知非继承她的实验，并把模型整理成审判；他的学科方向与马恩相连，审判核心是马恩瞒报宜居信号、明知薇蕾斯只是短期滤光驻留窗口。
- 龙夕若从病历、材料、治疗失败与司晨停顿诊断限制层，负责解放司晨。她的生命观不是一开始就坚硬，而是在红霞自杀后的后悔中变硬。
- 发现链包括：牛顿棱镜、夫琅禾费/巴尔末光谱、勒维耶式隐藏质量推断、L4/L5 共轨、爱因斯坦/Kerr 时间、外层/内层表面差异、臭氧/滤膜表层线索、信息限制、权限伦理。
- 地图发现链包括：教室滤光窗的红色偏移、东北谱线台的 H-alpha 异常、红河/红霞渡的强反照、根廊权限日志、主船舰桥传感器剪裁记录和马恩先遣主控室审计链。
- 终局不宜大段解释设定名词。更好的写法是让顾知非说案件层面的结论：马恩不知道黑洞，所以第一次说谎还以为有退路；但他知道薇蕾斯不是长期开放家园，仍然让检测器替未来背书。

## 天文定案

- 薇蕾斯是 Edmunds' Planet 式岩石星球，直接绕近极端 Kerr 超大质量黑洞 `KX-998` 运行，并与 B7-B8e / B8 IVe 型照明星 `BN-7 / Balmer-7` 处在同一黑洞圆轨道的近似 L4/L5 共轨结构中；黑洞负责时间膨胀，恒星从十余到几十 AU 量级提供蓝白偏紫照明。
- 终局可设置一次“合相/直列”式投影事件：稳定构型本体仍是 L4/L5 三角共轨，但从殖民地观测、透镜和光谱相位上，`KX-998`、`BN-7`、薇蕾斯会短暂压成一条可识别的证据线。
- 采用约 10.7 倍时间膨胀，使主角团本地 28 岁时外界约过 300 年。
- 紫色日光来自蓝白高 UV 恒星、大气散射/再辐射和 Be 星发射线；红霞由 Be 星盘、H-alpha 和大气筛分共同触发。
- 70 年窗口不再定义为人物健康风险，而是外层地表、内层滤光防护、开放地表活动和殖民维护共同失去余量的工程窗口；`BN-7` 高 UV、粒子事件、磁层不足和臭氧/滤膜脆弱性保留为表层环境压力与红霞线索。
- 薇蕾斯最好设在固定受光的“亮弯带”：`BN-7` 长期可见，`KX-998` 压在地平线下或禁区方位。

## 主旨

故事不应写成“小朋友打败邪恶成年人”的童话。马恩、安泊、方舟各自代表一种人类失败：瞒报、放弃、以抽象生存之名牺牲具体的人。

人类延续主题已确认：顾知非的核心感叹是“这样一个注定毁灭的世界不值得认真生活”。这不是轻率厌世，而是对马恩和方舟的判决：一个把短期窗口命名为未来，一个把困难世界命名为失败分支。
